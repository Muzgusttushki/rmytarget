    Авторизация для работы с API MyTarget с помощью R клиента rmytarget

    Ваш Токен доступа:

    Введите значение данного токена в R консоль в качестве ответа на запрос "Enter your token: ", после чего в R будет создан объект содержащий токен доступа к API MyTarget и вы сможете получить данные с помощью функций доступных в пакете ryandexdirect.

    Подробное описание всех доступных функций пакета можно найти тут.
